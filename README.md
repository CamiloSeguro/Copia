# Sistema de Préstamo de Recursos  
### Ecosistema de Laboratorios – Escuela de Artes y Humanidades  
**Universidad EAFIT**

---

##  Descripción

El **Sistema de Préstamo de Recursos** es una plataforma digital diseñada para optimizar la gestión, solicitud y seguimiento de equipos y recursos dentro del Ecosistema de Laboratorios de la Escuela de Artes y Humanidades.

Este proyecto busca:

- Centralizar la gestión de préstamos  
- Mejorar la trazabilidad de equipos  
- Optimizar la experiencia de estudiantes, docentes y personal administrativo  
- Reducir errores en registros manuales  
- Generar métricas para toma de decisiones  

---

##  Problema

Actualmente, la gestión de préstamos puede presentar:

- Procesos manuales o dispersos  
- Baja trazabilidad en tiempos de préstamo  
- Dificultad para consultar disponibilidad en tiempo real  
- Falta de métricas consolidadas  

Este sistema propone una solución digital estructurada y escalable.

---

##  Objetivos

### Objetivo general

Diseñar e implementar un sistema digital para la gestión eficiente de recursos en préstamo dentro del Ecosistema de Laboratorios.

### Objetivos específicos

- Permitir solicitudes digitales de préstamo  
- Visualizar disponibilidad en tiempo real  
- Registrar historial de préstamos  
- Generar reportes y métricas de uso  
- Alinear la interfaz con el Manual de Marca EAFIT  

---
